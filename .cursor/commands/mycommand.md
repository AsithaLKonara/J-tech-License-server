PROJECT COMMANDS (What Cursor Is Allowed To Do)

These are explicit permissions for Cursor actions.

✅ Allowed

Refactor for clarity and maintainability

Add types, validation, and guards

Improve accessibility and UX consistency

Introduce tests and mocks

Optimize performance only if measurable

❌ Not Allowed

Large rewrites without request

Opinionated framework swaps

Breaking API changes without warning

Over-engineering

⚠️ Conditional

Change architecture → require approval
