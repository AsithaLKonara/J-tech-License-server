The **J-Tech Pixel LED Upload Bridge** is a pro-grade platform for designing, simulating, and uploading patterns to Pixel LED matrices. It combines a high-performance Python desktop experience with a secure Laravel-powered license and subscription backend.

---

## üåü Key Features

- **Advanced Design Engine**: Multi-stop linear/radial gradients, shape constraints, and global bucket fill.
- **100% Parity Simulation**: Preview exact hardware behavior (including `scrollText` and rotations) before uploading.
- **Enterprise Licensing**: Account-based subscriptions with 30-day offline grace periods and hardware-bound tokens.
- **Single-Click Upload**: High-speed WiFi and local preview support for rapid prototyping.

## üöÄ Quick Start

### For Designers
1. Download the latest `UploadBridge.exe`.
2. Activate your license using your account.
3. Start painting and click **Upload and Run**.

### For Developers
1. Clone the repo: `git clone --recursive [URL]`
2. Install dependencies: `pip install -r apps/upload-bridge/requirements.txt`
3. Run: `python apps/upload-bridge/main.py`
