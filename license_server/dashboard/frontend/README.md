# Dashboard Frontend

Basic HTML/JavaScript dashboard for Upload Bridge account management.

## Features

- Device management (list, rename, revoke)
- Billing information display
- License/entitlement status
- Account settings

## Setup

1. Serve the dashboard files through the Express server
2. Configure Auth0 authentication
3. Update API_BASE URL in dashboard.js if needed
