# License Server Dashboard Frontend

Simple web dashboard for the Upload Bridge License Server.

## Features

- ✅ User login with email/password
- ✅ Display user information
- ✅ Show license plan and features
- ✅ Session management
- ✅ Responsive design

## Usage

The dashboard is automatically served at the root URL when deployed to Vercel:
- `https://j-tech-license-server.vercel.app/`
