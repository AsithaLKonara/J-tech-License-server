### The Design Canvas
- **Brush & Pen**: Core tools for drawing individual pixels or strokes.
*   **Eraser**: Use to clear specific pixels.
- **Shapes**: Create perfect Lines, Rectangles, and Circles (Hold `Shift` to constrain ratio).
- **Gradients**: Select the Gradient tool and drag on the canvas to apply multi-stop linear or radial gradients.
- **Bucket Fill**: Fill enclosed areas. Uncheck "Contiguous" in the settings to replace all instances of a color globally.

### Selection & Manipulation
- **Marquee Selection**: Drag a box to select an area.
- **Move**: Click and drag inside a selection to move pixels.
- **Copy/Paste**: Use `Ctrl+C` and `Ctrl+V` to duplicate selections.
