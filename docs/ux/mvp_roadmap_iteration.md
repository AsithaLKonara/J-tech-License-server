# MVP Roadmap, Beta & Iteration Plan

## Phased Delivery

1. **Timeline MVP**
   - Implement new timeline UI and basic playback.  
   - Integrate with FrameManager/history.  
   - Success: 20% faster frame edit workflow.

2. **Playback & Layer Enhancements**
   - Advanced transport controls, layer inspector refinements, automation overlays.  
   - Success: 80% beta satisfaction ≥4/5.

3. **Theming & Polish**
   - Apply visual theme, accessibility improvements, micro-interactions.  
   - Success: SUS ≥75, fewer UI support tickets.

## Beta Program

- Invite 15–20 power users; provide onboarding and change log.  
- Weekly feedback sync; track qualitative notes.

## Release Communication

- Blog/tutorial, in-app “What’s New”, keyboard shortcut cheat sheet.  
- Feature flag fallback to legacy UI during transition.

## Iteration Loop

1. Collect telemetry + feedback weekly.  
2. Prioritise issues (severity, frequency).  
3. Bi-weekly sprints for refinements.  
4. Quarterly design review to reset roadmap.

## Success Metrics Dashboard

- Timeline usage (frames edited/session).  
- Playback engagement.  
- Preset usage/export frequency.  
- NPS for new experience.
